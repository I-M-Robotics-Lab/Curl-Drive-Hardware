# Curl-Drive: BLDC Motor Controller

This repository contains the hardware PCB design, schematics, and production files of the brushless motor FOC controller **Curl-Drive** developed by I'M Robotics Lab. 

The firmware is still under development, but the hardware has been validated with no major issues. Full version is planned to be released in early December. 

We welcome contributions from anyone! To get involved or ask questions, join our Discord server.
